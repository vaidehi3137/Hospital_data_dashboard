# Hospital Patient Data Dashboard

An Excel-based dashboard analyzing hospital patient admissions, demographics, and satisfaction metrics to support healthcare operations insights.

## 📊 Overview

This project processes patient-level hospital data (9,000+ records) into a summarized dashboard, helping identify admission trends, department referral patterns, and patient satisfaction/wait-time performance.

## 🗂️ Dataset

The dataset includes:

- **Patient Id** — unique patient identifier
- **Patient Admission Date / Time / Timeline** — admission timing details
- **Patient Gender / Age / Age Range / Race** — demographic fields
- **Department Referral** — department the patient was referred to
- **Patient Admission Flag** — admission status indicator
- **Patient Satisfaction Score** — patient-reported satisfaction
- **Patient Waittime** — time waited before being seen

## 🛠️ Structure

The workbook has 3 sheets:

1. **Sheet1** — raw patient-level data
2. **Data sheet** — summarized/aggregated data
3. **Dashboard** — consolidated visual dashboard for quick insights

## 🔍 Key Analysis Areas

- Admission trends by department and time
- Patient demographics breakdown (age, gender, race)
- Satisfaction score vs. wait time analysis
- Department referral patterns

## 🧰 Tools Used

- Microsoft Excel (Pivot Tables, Pivot Charts, Dashboard design)
